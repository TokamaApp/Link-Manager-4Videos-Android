Android app for structured URL list management with import, rating, filtering, sorting and multi-browser opening. Published publicly for transparency and source inspection only.

## Play Store

[View on Google Play](https://play.google.com/store/apps/details?id=com.tokama.linkmanager)

## Features

- Create, import, rename and manage custom TXT-based link lists
- Detect and extract links directly from free text
- Open single or multiple selected links at once
- Rate links from **Very good** to **Very bad**
- Filter by **seen / unseen**, **rating** and **sorting**
- Reorder files and links via drag and drop
- Select and delete multiple links at once
- German and English app language support
- App theme support with **Night / Day / Auto**

## Supported Browsers

The app can open links in supported browsers such as:

- Chrome
- Firefox
- Brave
- Microsoft Edge
- Samsung Internet
- Vivaldi
- Opera
- DuckDuckGo Browser
- Tor Browser

This makes it useful for workflows where links should be opened intentionally in different browsers depending on purpose.

## Use Cases

Link-Manager: 4Videos is intended for structured link workflows, including:

- video collections
- research links
- downloads
- project link sets
- personal collections
- task-based URL processing
